## How to run

1. Create and activate a virtual environment (optional but recommended):

  bash
  python3 -m venv .venv
  source .venv/bin/activate
  python -m pip install -U pip


2. Install dependencies:

   python -m pip install numpy pandas matplotlib seaborn scikit-learn imbalanced-learn


3. Make sure the dataset file bank-additional.csv is in the same folder as the notebook (the repository root ML-fundamentals-2026).

4. Open the notebook:

   jupyter notebook assignment_1_viktoriia_stepanenko.ipynb


5. In the notebook, select the Python interpreter from .venv (if used), then run Restart Kernel and Run All to execute every cell. The notebook expects bank-additional.csv (the smaller UCI Bank Marketing
dataset) and will reproduce all steps of the data preparation pipeline and logistic regression sanity check described in the assignment.
